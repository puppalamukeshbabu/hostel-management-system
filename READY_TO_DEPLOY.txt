📋 THREE SIMPLE STEPS TO DEPLOY:

1. PUSH TO GITHUB (5 min)
   git init && git add . && git commit -m "Deploy ready"
   git remote add origin https://github.com/YOUR_USERNAME/hostel-ms.git
   git push -u origin main

2. SET UP MONGODB (10 min)
   - Visit https://www.mongodb.com/cloud/atlas
   - Create free cluster and get connection string
   - Copy: mongodb+srv://user:pass@cluster...

3. DEPLOY ON RENDER (5 min)
   - Visit https://render.com
   - New Web Service → GitHub repo
   - Build: npm run setup
   - Start: npm start
   - Add environment variables (NODE_ENV, MONGODB_URI)
   - Deploy!
